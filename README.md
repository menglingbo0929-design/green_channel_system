# 高校绿色通道系统

高校绿色通道管理系统采用前后端分离架构开发，用于完成新生信息管理、绿色通道申请、爱心礼包申请、三级审核、欠费确认、申请补录和统计报表等业务。

## 技术栈

### 前端

- Vue 3
- Vite
- Vue Router
- Pinia
- Axios
- Element Plus
- ECharts

### 后端

- Spring Boot
- MyBatis
- MySQL
- Maven
- JWT
